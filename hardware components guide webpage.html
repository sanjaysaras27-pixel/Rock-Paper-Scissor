Hardware Components

CPU

The Central Processing Unit (CPU) is the primary component and "brain" of a computer, responsible for interpreting and executing instructions from software and hardware. Located on the motherboard, this chip processes data, performs calculations (ALU), and manages system operations (control unit). Modern CPUs often feature multiple cores, enabling rapid, simultaneous multitasking and high-speed processing.

RAM

>Random Access Memory (RAM) is a computer's high-speed, short-term, volatile memory used to temporarily store data for active applications and the operating system, allowing the CPU to access information rapidly. As the primary workspace for a computer, more RAM enables better multitasking and faster system performance. Data in RAM is erased when power is turned off.
